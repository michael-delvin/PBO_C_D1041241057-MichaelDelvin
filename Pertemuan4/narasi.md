1. Naming Convention adalah metode penamaan sebuah variabel supaya variabel itu mudah dipahami atau bisa kita temukan jika kita melupakan nama variabelnya berdasarkan konteksnya
   atau fungsinya dengan aturan-aturan tertentu sebagai contoh jika sebuah variabel memiliki dua kata atau lebih maka pada kata kedua dan seterusnya tanpa spasi harus diawali oleh
   huruf kapital, jika sebuah variabel hanya memiliki satu kata maka kata tersebut diawali oleh huruf kapital. Pada tipe data boolean sebaiknya diawali dengan pertanyaan dan 
   disusul oleh kata kerja berdasarkan fungsinya.
2. Variabel adalah tempat kita untuk menyimpan sebuah data yang kita inputkan ke dalam sebuah program, penamaan variabel sangat penting karena penamaan itu berfungsi sebagai
   identitas dalam mengakses ataupun memanipulasi data dalam program. Class digambarkan seperti sebuah blueprint atau templat untuk menciptakan objek, yang mendefinisikan variabel
   dan fungsi yang akan dimiliki oleh class tersebut, secara sederhana class adalah sebuah dasar dalam program untuk kita membuat data, mengolah data itu dan melakukan 
   pengembangan program berdasarkan data variabel yang dimiliki.
3. Operator aritmatika digunakan untuk melakukan operasi matematika dasar yaitu penjumlahan (+), pengurangan (-), perkalian (*), pembagian (/), dan modulus (%). Operator-operator
   ini bekerja dengan cara mengambil satu atau dua nilai variabel untuk menghasilkan nilai baru dengan operan yang dimiliki oleh operator aritmatika atau operasi matematika dasar.
4. 1) Latihan p022WorkingVariable merupakan latihan untuk mengubah nilai variabel yang telah ada.
   2) Latihan p023PrimitiveData merupakan latihan untuk mengetahui tipe primitive data yang ada pada bahasa pemrograman Java.
   3) Latihan p041NamingConvention merupakan latihan untuk penamaan sebuah variabel yang sesuai dengan aturan-aturan tertentu.
   4) Latihan p042OperatorAritmatika merupakan latihan untuk melakukan perhitungan dengan operator aritmatika yang digunakan pada Java.
   5) Latihan LatihDataMahasiswa merupakan latihan menggunakan berbagai tipe data yang telah dipelajari untuk menampilkan Data Mahasiswa.
   6) Latihan LatihPerhitungan merupakan latihan untuk menggunakan Compound Assigntment seperti += dan -= untuk penjumlahan dan pengurangan suatu data.
   7) LatihanProduk merupakan latihan untuk mendeskripsikan sebuah produk dengan menerapkan NamingConvertion dan Operator Aritmatika berdasarkan data yang sudah diberikan.
   8) LatihanKalkulator merupakan latihan penerapan Operator Aritmatika dengan membuat sebuah program Kalkulator Sederhana dengan menggunakan operasi matematika dasar.
   9) LatihanCounter merupakan latihan lanjutan untuk penerapan Compound Assignment sebuah nilai dengan operasi matematika dasar yang telah dipelajari.
